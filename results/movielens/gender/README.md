# movielens / gender result summaries

This folder stores summary files for the `movielens` domain and `gender`
counterfactual attribute setting.

Recommended files:

- main_metrics_summary_*.json
- quadrants_joint_otsu_summary_*.json
- layerwise_internal_summary_*.json
- match10_summary_*.json

Full raw generations and hidden-vector files are intentionally excluded from the
anonymous artifact due to size and compute requirements.
